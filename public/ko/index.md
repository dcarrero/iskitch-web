# iSkitch — macOS용 캡처, 주석, 공유 앱

> iSkitch는 클래식 Skitch를 현대적으로 재해석한 macOS 네이티브 앱입니다. 영역, 창, 전체 화면을 캡처하고 즉시 주석을 달아 클릭 한 번으로 공유하세요.

**상태:** Mac App Store 출시 예정. 비공개 베타 곧 시작.
**플랫폼:** macOS 14 이상. Universal Binary (Apple Silicon + Intel).
**언어:** 한국어, English, Español, Français, Deutsch, Italiano, Português, 日本語.
**가격:** 베타 기간 무료.

## iSkitch란

iSkitch는 클래식 Skitch를 현대적으로 재해석한 macOS 네이티브 앱입니다. 메뉴 막대에 상주하며 언제든 ⌥⌘4로 호출 — macOS 공유 시트로 바로 공유합니다.

스페인 마드리드에 본사를 둔 [Color Vivo Internet, S.L.](https://colorvivo.com)이 SwiftUI + AppKit으로 개발했습니다.

## 기능

- **무엇이든 캡처.** 영역 (⌥⌘4), 창, 전체 화면.
- **14가지 주석 도구.** 테이퍼드 화살표, 시스템 글꼴 텍스트, 도형, 형광펜, 펜, 스탬프 등.
- **민감 정보 가리기.** 픽셀화, 흐림, 솔리드 블록, 줄무늬.
- **네이티브 공유.** macOS 공유 시트, 어떤 앱으로든 드래그, PNG/JPG/PDF 복사·저장.
- **스포트라이트와 자르기.**
- **프라이버시 우선.** 계정 없음, 추적 없음, 데이터는 Mac을 떠나지 않음.
